An interactive Streamlit application that visualizes real-time traffic flow and incidents using the TomTom Traffic API. Explore traffic conditions across major cities worldwide with an intuitive map interface featuring multiple base map styles and customizable overlay layers.

Simply enter your TomTom API key (get one free at developer.tomtom.com) to access live traffic flow data and incident reports. The app offers pre-configured locations including Istanbul, Ankara, Berlin, New York, Tokyo, and more, with adjustable layer opacity, multiple map themes, and interactive controls for a seamless traffic monitoring experience.
